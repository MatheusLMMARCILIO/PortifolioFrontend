{ Minhas Habilidades }
HTML

HTML é a linguagem usada para criar e estruturar páginas da web. Ela define a estrutura e o conteúdo dos sites usando tags e elementos.

CSS

CSS é a linguagem usada para estilizar e formatar páginas da web. Ela define a aparência visual dos sites, como cores, fontes e layout, aplicando estilos aos elementos HTML.

JavaScript

JavaScript é uma linguagem de programação usada para criar interatividade e dinamismo em páginas da web. Ela permite a manipulação do conteúdo HTML e CSS, além de responder a eventos e executar funcionalidades no navegador.

React

React é uma biblioteca JavaScript para construir interfaces de usuário interativas. Ela facilita a criação de componentes reutilizáveis e a gestão do estado da aplicação
